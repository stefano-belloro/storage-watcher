Storage Watcher

A web application that tests your browser's support of modern client-side storage techniques and its privacy settings

Test 1 - API support: Some browsers disable certain APIs when used in private browsing mode. Simply try to add an entry to the storage system by pressing 'Add Entry' and click on 'List all entries'.

Test 2 - Data deletion: After completing test 1, remove your personal data from your browser, then click on 'List all entries'. All storage should be empty, but if any entry is still there it means that your browser did not actually delete locally stored data.
